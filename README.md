LOTM – The Blasphemy Project

All pathways Lord of the Mysteries (book 1)!

LOTM – The Blasphemy Project is a visual and experimental frontend project inspired by the the web novel Lord of the Mysteries.

The main goal of this project is to explore advanced CSS 3D transforms, modern layout techniques, and smooth interactions using React without relying on heavy animation libraries.

The interface is built as a living scene rather than a static page:

This project focuses on clean structure, visual impact, and performance, combining minimalism with depth to create an immersive experience. It is designed both as a creative experiment and as a technical showcase of modern frontend capabilities.
